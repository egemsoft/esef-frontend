refresh
service in module esef.frontend.refresh

Description

Refresh service implements a recursive periodical refresh function. Provides easy to use simple API to run a function periodically by given interval.

Methods

  • isStarted()

    Returns true if refresh function is running; false if it is not.

    Returns
    object
    • Refresh status true if it is started.

  • isStopped()

    Returns true if refresh function is not running; false if it is.

    Returns
    object
    • Refresh status true if it is stopped.

  • setInterval(interval)

    Sets interval for refresh timeout which is set 10000 as default.

    Parameters
    ParamTypeDetails
    intervalnumber
    • Interval in terms of milliseconds.
    Returns
    object
    • refresh service to support chaining.

  • start(callback)

    Main method which starts refreshing. Takes callback param that will be run on every period.

    Parameters
    ParamTypeDetails
    callbackfunction
    • Callback function to be run periodically.
    Returns
    object
    • refresh service to support chaining.

  • stop()

    Stops refresh function if it is started.

    Returns
    object
    • refresh service to support chaining.
